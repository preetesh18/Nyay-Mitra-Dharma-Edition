# ⚖️ Dharma Nyaya — धर्म न्याय
### The Ancient Oracle of Dharmic Justice

A Python Flask web application powered by **Gemini 2.5 Flash** that delivers unbiased judicial verdicts grounded in the eternal wisdom of Hindu scriptures, Sanskrit texts, and ancient Indian societal knowledge.

---

## ✨ Features

- **Scriptural Analysis** — Draws from 15+ ancient texts including Manusmriti, Arthashastra, Bhagavad Gita, Mahabharata, Upanishads, and more
- **Original Shlokas** — Provides verses in Devanagari/Sanskrit script with full transliteration
- **Deep Citations** — Every verdict includes source reference, translation, and contextual meaning
- **Unbiased Verdicts** — Dharma-guided analysis of plaintiff, defendant, and case facts
- **Remedies** — Prescribes dharmic remedies and moral teachings alongside the verdict
- **Stunning UI** — Ancient Indian manuscript aesthetic with gold, saffron, and deep ink tones

---

## 🚀 Setup & Run

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the application

```bash
python app.py
```

### 3. Open in browser

```
http://localhost:5000
```

### 4. Enter your Gemini API key

Get your free API key from: https://aistudio.google.com/app/apikey
